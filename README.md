# SpeedMath

**SpeedMath** is a fast-paced, 1v1 local multiplayer arithmetic game developed by **FurtunaLabs**. Designed to test both your mathematical accuracy and reaction time, players compete head-to-head to solve equations before their opponent does.

---

## ⚡ Core Mechanics
* **Speed is Key:** The first player to select the correct answer scores a point.
* **Zero Tolerance:** Selecting a wrong answer instantly awards a point to your opponent.
* **Victory Condition:** The first player to reach **10 points** wins the match.

---

## 🕹️ Controls (Local 1v1)

| Player | Action Keys |
| :--- | :--- |
| **Player 1** | `A` `S` `D` |
| **Player 2** | `←` `↓` `→` |

---

## 🛠️ Installation & Usage
This application is built using **C# WinForms**. To run the game on your machine:

1. Clone this repository or download it as a ZIP file.
2. Navigate to the executable directory: `proiect/bin/Debug`.
3. Run the `SpeedMath.exe` file.

*(Note: Ensure you have a compatible Windows environment to run .NET/WinForms applications).*
